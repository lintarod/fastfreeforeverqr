QR Code Size Guide: How Small Can You Print?

5 min read · Updated January 2025

Home › Blog › QR Code Size Guide

One of the most common questions about QR codes is: How small can I print it? The answer depends on several factors, including where it will be used, how far away people will scan from, and how much data is encoded.

This guide covers everything you need to know about QR code sizing for different applications.

The Golden Rule: 10:1 Ratio

The industry-standard rule is the 10:1 scanning distance ratio. For every 10 units of distance, you need 1 unit of QR code size.

  • Scanning from 10 inches away? QR code should be at least 1 inch.
  • Scanning from 10 feet away? QR code should be at least 1 foot.

This is a conservative estimate. Modern phones can often scan smaller codes, but the 10:1 ratio ensures reliability across all devices.

Factors That Affect Minimum Size

1. Data Density

The more data encoded, the more complex the QR code becomes. A QR code containing a short URL like "bit.ly/abc" can be printed smaller than one containing a long URL with tracking parameters.

Pro Tip: Use a URL shortener to reduce QR code complexity and allow for smaller printing.

2. Error Correction Level

QR codes have four error correction levels: L (7%), M (15%), Q (25%), and H (30%). Higher error correction means the code can still scan even if partially damaged, but it also increases complexity. For small prints, use Level L or M.

3. Print Quality

High-resolution printing allows for smaller codes. Newspaper printing or low-quality inkjet may require larger sizes than professional offset or digital printing.

4. Material Surface

Glossy surfaces can cause glare. Textured surfaces may distort the code. Curved surfaces (like bottles) need larger codes to compensate for distortion.

The Quiet Zone

Every QR code needs a "quiet zone" — white space around the code that helps scanners identify its boundaries. The quiet zone should be at least 4 modules (the small squares that make up the code) wide.

Don't let text, graphics, or borders touch the QR code. This is one of the most common causes of scanning failures.

Testing Is Essential

Before printing in bulk, always:

  1. Print a test at the intended size
  2. Test with multiple phones (iPhone, Android, old and new models)
  3. Test in the actual lighting conditions where it will be used
  4. Test from the expected scanning distance

SVG: The Best Format for Printing

When downloading your QR code, choose SVG format if you need to resize it. SVG is a vector format that scales infinitely without losing quality. PNG and JPEG are raster formats that become blurry when enlarged.

Summary

  • Absolute minimum: 0.4" × 0.4" (10mm) for close-up scanning
  • Safe minimum for most uses: 0.8" × 0.8" (20mm)
  • Use the 10:1 ratio for distance scanning
  • Always include quiet zone around the code
  • Use shorter URLs and lower error correction for smaller codes
  • Download as SVG for scalable printing
  • Test before bulk printing
